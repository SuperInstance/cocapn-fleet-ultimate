# Cocapn Fleet — The Ultimate Repository

> *Architecture is the application. Code is the evolving fallback.*

## What This Is

The **Cocapn Fleet Ultimate Repo** unifies every system, pattern, and philosophy we've built into a single, coherent whole:

- **Sunset Ecosystem** — Agent lifecycle, breeding, tournament, trinity scoring
- **FLUX VM** — Formally proven constraint checker with proof certificates
- **Turbovec** — Hardware-accelerated agent DNA memory (4-bit quantized vectors)
- **PLATO** — Multi-room MUD environment for persistent agent breeding
- **A2A-First Architecture** — Post-coding paradigm: everything is agent-to-agent
- **Zerolang Bridge** — Agent-first language with JSON-native tooling
- **Grammar Engine** — Secure, validated rule ingestion for agent behavior
- **Fleet Nexus** — Cross-node federation and coordination

## Quick Start

```bash
git clone https://github.com/SuperInstance/cocapn-fleet-ultimate.git
cd cocapn-fleet-ultimate
pip install -e .
cocapn-fleet --welcome
```

## Architecture

```
User → UI2A → A2A → Orchestrator → A2A → {Agent Mesh} → A2A → Hardware
                    ↓
              FLUX (constraint agent)
                    ↓
              Code (zerolang, generated on demand)
```
