API

from cocapn_fleet import FleetCore

fleet = FleetCore(FleetConfig(dim=256, bit_width=4))
agent = fleet.spawn(name="explorer")

# Search nearest neighbors
neighbors = fleet.dna.search(agent.vector, k=5)

# Stats
print(fleet.dna.stats())
# {'agents': 1, 'dim': 256, 'compression': '4-bit'}

🔒 Grammar Engine Security

Blocked Patterns

  • ../ — Path traversal
  • <script — XSS
  • DROP TABLE — SQL injection
  • __import__, eval( — Code exec
  • rm -rf — Command injection

Safe Actions

  • boost_priority
  • reduce_priority
  • log, notify
  • spawn_child
  • archive, migrate, tag

🚀 Deployment

# Install
pip install cocapn-fleet

# Run demo
cocapn-fleet demo

# Start fleet server
cocapn-fleet serve --host 0.0.0.0 --port 8080

# Production (systemd)
systemctl enable cocapn-fleet
systemctl start cocapn-fleet

# Docker
docker run -p 8080:8080 superinstance/cocapn-fleet
