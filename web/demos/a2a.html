Task Delegation Flow

1
Discover
Client fetches Agent Card from /.well-known/agent.json
2
Send Task
POST to /tasks/send with task specification
3
Process
Agent processes task, generates artefacts
4
Receive Result
JSON response with status, artefacts, and proof certificates
